# Data Requirements

Raw data are not included in this repository.

## Reaction Wheel Data

```text
data/raw/附件1 reaction_wheel_3500d_data.csv
data/raw/附件2 reaction_wheel_1800d_data.csv
```

Or set:

```bash
export MM_CONTEST_ATT1=/path/to/attachment1.csv
export MM_CONTEST_ATT2=/path/to/attachment2.csv
```

## Bearing Data

```text
data/raw/XJTU-SY_Bearing_Datasets/
```

Or set:

```bash
export XJTU_SY_ROOT=/path/to/XJTU-SY_Bearing_Datasets
```
