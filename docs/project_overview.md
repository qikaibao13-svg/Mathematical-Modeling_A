# Project Overview

The project contains a four-task workflow for reaction wheel health assessment.

- Task 1 establishes the reaction wheel baseline.
- Task 2 constructs transferable bearing degradation representations.
- Task 3 performs baseline-constrained transfer residual correction.
- Task 4 converts model outputs into an engineering warning report.

The current Attachment 2 result is a Level 2 yellow warning: the flywheel is close to the decline boundary, but the equivalent RUL remains above the short-term severe-warning threshold.
