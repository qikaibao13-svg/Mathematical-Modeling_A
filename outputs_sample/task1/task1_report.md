# 任务一：反作用飞轮退化建模与健康状态初评估结果

## 1. 数据检查与变量构造

附件1为仿真全寿命数据，共 176 条记录，时间范围为 0-3500 天，采样间隔为 20 天；附件2为在轨监测数据，共 1801 条记录，时间范围为 0-1800 天，采样间隔为 1 天。两份数据均无缺失值、无重复 day。`speed_rpm` 在两份数据中均恒为 3000 rpm，因此本任务一中不作为退化变量。

按方案构造变量：

- `I_obs = current`
- `I_theo = current_theoretical`
- `T = temperature`
- `F = friction_torque`，仅附件1可用
- `delta_I = current - current_theoretical`

附件1中，`current` 由 0.1137 A 上升到 1.6139 A，增幅约 1319.44%；`current_theoretical` 由 0.109702 A 上升到 1.607245 A，增幅约 1365.10%；`temperature` 由 35.0 上升到 47.0；`friction_torque` 由 0.004498 上升到 0.065897，增幅约 1365.03%。上述四个退化相关变量与时间的 Spearman 相关系数均接近或等于 1，说明全寿命退化趋势显著。`delta_I` 均值接近 0，主要反映观测扰动和局部偏差，不宜作为主退化变量。

附件2中，`current` 由 0.1137 A 上升到 1.0088 A，`current_theoretical` 由 0.1097 A 上升到 0.993598 A，`temperature` 由 35.0 上升到 42.2，均呈明显上升趋势。附件2末端平滑后当前值为：`current_s = 0.979329 A`，`current_theoretical_s = 0.980733 A`，`temperature_s = 41.925806`。

## 2. 平滑与健康指数

为降低观测噪声并保留长期退化趋势，对 `current`、`current_theoretical`、`temperature` 采用居中滑动平均：

- 附件1：9 点窗口，对应约 180 天；
- 附件2：61 点窗口，对应约 61 天。

该处理符合方案中“滑动平均或 Savitzky-Golay”的要求。考虑到本任务以长期退化建模为主，滑动平均更便于解释，且不会引入额外模型假设。

归一化采用附件1全寿命范围作为基准，使附件2可以映射到同一健康尺度。健康指数按方案初值取：

- 附件1：`HI1 = 0.45 norm(I_obs) + 0.20 norm(T) + 0.35 norm(F)`
- 附件2：`HI2 = 0.65 norm(I_obs) + 0.25 norm(T) + 0.10 norm(|delta_I|)`

权重未作微调。理由是：附件1的 `HI1` 从 0 单调上升到 1，Spearman 相关系数为 1.0，能够稳定表征全寿命退化；附件2的 `HI2` 虽受 `delta_I` 日尺度扰动影响，但总体随时间上升，Spearman 相关系数约为 0.988，满足当前任务的健康状态评估要求。

## 3. 模型 A：平移指数退化模型

模型形式为

`y(t)=c+a(exp(bt)-1)`。

在附件1上拟合结果如下：

| 对象 | c | a | b | R2 | RMSE | MAE |
|---|---:|---:|---:|---:|---:|---:|
| current_theoretical | 0.109512 | 0.633688 | 0.00034604 | 0.999954 | 0.002907 | 0.001245 |
| friction_torque | 0.004498 | 0.025539 | 0.00035000 | 1.000000 | 0.00000028 | 0.00000024 |
| HI1 | -0.006550 | 0.429792 | 0.00034703 | 0.999963 | 0.001772 | 0.001087 |

指数模型对 `current_theoretical`、`friction_torque` 和 `HI1` 均具有极高拟合优度，且参数 b 量级一致，说明电流理论趋势、摩擦力矩和综合健康指数具有一致的加速退化特征。

伪在线检验中，仅使用附件1前 60%、70%、80% 数据预测最终失效日，得到失效日分别为 3447.13、3453.72、3456.99 天，相对真实全寿命终点 3500 天偏早约 43-53 天。说明指数模型具备较强外推能力，但早期截断条件下仍存在保守偏差。

## 4. 模型 B：三阶段分段线性模型与变化点

采用连续三段线性模型，并约束三段斜率非递减。以 `HI1` 为主进行阶段划分，得到：

- 第一变化点：`tau1 = 1340` 天；
- 第二变化点：`tau2 = 2500` 天；
- `HI1(tau1) = 0.2491`；
- `HI1(tau2) = 0.5874`。

据此划分：

- 健康期：0-1340 天，HI1 约为 0-0.249；
- 退化期：1340-2500 天，HI1 约为 0.249-0.587；
- 衰退期：2500-3500 天，HI1 大于约 0.587。

三段 HI1 斜率分别为 0.0001878、0.0002922、0.0004233 / 天，斜率随阶段递增，符合“健康期低速退化、退化期中速退化、衰退期高速退化”的工程解释。作为对照，在 `current` 上检测得到变化点为 1320 天和 2500 天，与 HI1 基本一致。

## 5. 附件2健康状态与 RUL

附件2末端 day=1800 的观测状态为：

- 原始 `current = 1.0088 A`；
- 平滑 `current = 0.979329 A`；
- 平滑 `current_theoretical = 0.980733 A`；
- 平滑 `temperature = 41.925806`；
- `delta_I = 0.015202 A`；
- `HI2 = 0.600085`。

路径一：模型逆映射。将附件2末端 `current_theoretical_s` 代入附件1指数模型，得到等效寿命位置：

- `t_inv = 2499.51` 天；
- `RUL_inv = 1000.49` 天；
- 残差自助法 95% 区间约为 `989.44-1011.42` 天。

路径二：模板轨迹匹配。以附件2末端最近窗口的 `current + temperature + HI` 轨迹匹配附件1全寿命模板：

- 90 天窗口：等效寿命 2481 天，RUL 1019 天；
- 180 天窗口：等效寿命 2464 天，RUL 1036 天；
- 365 天窗口：等效寿命 2423 天，RUL 1077 天。

主窗口取 180 天，得到 `t_match = 2464` 天，`RUL_match = 1036` 天。综合模型逆映射、模板匹配和附件1采样间隔不确定性，最终给出：

- RUL 点预测：约 `1000` 天；
- RUL 区间预测：约 `[979, 1125]` 天。

由于 `t_inv = 2499.51` 非常接近第二变化点 2500 天，而模板匹配结果位于 2464 天，因此当前飞轮应判定为“退化期后段，接近衰退期边界”，而不是简单写成稳定健康期或已经明显衰退期。

## 6. 数据验证结论与假设性判断

已由数据验证的结果：

- 两份附件均无缺失和重复时间点；
- 附件1中 `current`、`current_theoretical`、`temperature`、`friction_torque` 均随时间显著上升；
- 指数模型能高精度拟合 `current_theoretical`、`friction_torque` 和 `HI1`；
- 三阶段分段线性模型在 `HI1` 和 `current` 上给出稳定的第二变化点 2500 天；
- 附件2当前等效寿命位置约在 2464-2499 天附近，两条路径均落在退化期后段。

仍带假设性的判断：

- RUL 计算以附件1的 3500 天作为全寿命失效端点；
- 附件2与附件1可通过同一退化模板映射；
- 区间预测主要反映模型残差、模板窗口选择和附件1采样间隔引入的不确定性，尚未包含真实在轨环境差异造成的全部不确定性。
